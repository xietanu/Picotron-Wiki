# store_metadata(filename,meta)

## Overview

`store_metadata` stores metadata, `meta`, at a file or directory, `filename`, merging with the original metadata.

Source: [source.lua](source.lua)

## Arguments

### filename

The path to the file or directory to store metadata in

### meta

The metadata to store in the file or directory

## Returns

This function does not return anything

## Possible Future

Ability to remove metadata items from the pre-existing metadata
