# pset(x, y, [color])

## Overview

Sets the pixel at `x`, `y` to the color with index `color`.

When `color` is not specified, the current draw color is used.

`x` and `y` are offset by the camera (see [`camera`](/picotron_api/functions/camera/main.md)).

Source: [source.lua](source.lua)

## Arguments

### `x`: number

The x position of the pixel.

### `y`: number

The y position of the pixel.

### `color`: number

The index of the color to set the pixel to.
