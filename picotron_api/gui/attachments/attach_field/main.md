# attach_field(el)

## Overview

Attaches a text field to the element.

Used for editing small amounts of text.

Source: [source.lua](source.lua)

## Expected attributes

### x

x-position of the element

### y

y-position of the element

### width

width of the element

### height

height of the element

## Expected event callbacks

### get() event callback

A function to handle getting the field's intial text

### set() event callback

A function to handle setting the field's text to the user input
