# cursor

## Overview

Sets the mouse cursor when this element is hovered over.

### Values:

Default value: nil

See [`window()`][/picotron_api/functions/window/main.md]

0: no cursor
1: default cursor
userdata u8: a custom cursor sprite.

System cursors are named, and can be requested using a string: “pointer”, “grab”, “dial”, “crosshair”
