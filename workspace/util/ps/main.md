# ps

## Overview

Lists all active processes and details

Identical to the about.p64 view of processes but a snapshot, rather than actively updating

Source: [source.lua](source.lua)

### Details shown

#### pid

PID of the process

#### name

Name of the process

#### cpu

CPU usage of the process (in decimal, rather than percentage)

#### pri

Priority of the process

#### mem

Memory usage of the process in Kilobytes
