# Map Editor > Keybinds

## Overview

Information from here is sourced from the [Picotron Manual](https://www.lexaloffle.com/dl/docs/picotron_manual.html)

This goes over all the keybinds built into the Map Editor for Picotron.

The map editor uses similar shortcuts to the @{GFX Editor}, with a few changes in meaning.

## Keybinds

### SPACE
Shortcut for the [Pan Tool](/environment/map_editor/tools/main.md)

### MOUSEWHEEL
Zoom in and out

### S
Shortcut for the [Select Tool](/environment/map_editor/tools/main.md) (hold down)

### CTRL-A
Select the entire canvas

### ENTER
Deselect everything

### ARROW KEYS
Move selection

### CTRL-C
Copy selection

### CTRL-V
Paste selection

### TAB
Toggle the RH pane

### -
Select previous sprite

### +
Select next sprite

### RMB
Set the active color to the color of the hovered pixel

### F
Flips selected tiles horizontally

### V
Flips selected tiles vertically

### R
Rotates selected tiles
