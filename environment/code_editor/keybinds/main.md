# Code Editor > Keybinds

## Overview

Information from here is sourced from the [Picotron Manual](https://www.lexaloffle.com/dl/docs/picotron_manual.html)

This goes over all the keybinds built into the Code Editor for Picotron.

## Keybinds

### CTRL-X
Cut the selected text

Copies the text to clipboard and then deletes it from the editor.

### CTRL-C
Copy the selected text to the clipboard

### CTRL-V
Pasted the selected text from the clipboard

### CTRL-Z
Undo the last change

### CTRL-Y
Redo the last change

### CTRL-F
Search for text in the current tab

### CTRL-L
Jump to a line number

### CTRL-W
Jump to the start of the current line

### CTRL-E
Jump to the end of the current line

### CTRL-D
Duplicate current line

### TAB
Indent a selection

### SHIFT-TAB
Unindent a selection

### CTRL-B
Comment / uncomment selected block of text

### SHIFT-ENTER
To automatically add block end and indent

### CTRL-UP/CTRL-HOME
Jump to the start of the file

### CTRL-DOWN/CTRL-END
Jump to the end of the file

### CTRL-CURSORS
Jump left or right by word
