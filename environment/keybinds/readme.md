## General

The `ALT` key is `OPTION` on Mac devices.

ALT+ENTER - Sets Picotron to be fullscreen (in the Host OS)

[ALT+F4](/environment/fastquit/main.md) - Fast Quit (Windows)

[CTRL+Q](/environment/fastquit/main.md) - Fast Quit (Mac, Linux) - requires enabling in settings

[CTRL+R](/environment/reload/main.md) - Reload / Run / Restart cartridge

[CTRL+SHIFT+R](/environment/reload/main.md) - Live-reload a single .lua or .gfx file while /ram/cart is running

[CTRL+S](/environment/save/main.md) - Quick save the working cartridge or the current file

[ALT+LEFT/RIGHT](/environment/workspaces/main.md) - Cycle workspaces

[ESCAPE](/environment/workspaces/main.md) - Toggle between the desktop and the terminal

[ENTER](/environment/pause/main.md) - 

[CTRL+6](/environment/screenshot/main.md) - Capture a screenshot (png) of the current screen, stored in `{Your Host OS desktop}/picotron_desktop`

[CTRL+7](/environment/screenshot/main.md) - Capture the label for the currently loaded cartridge

[SHIFT+CTRL+8](/environment/gif/main.md) - Select a region for the gif (default is the full visible screen)

[CTRL+8](/environment/gif/main.md) - Begin capturing a gif

[CTRL+9](/environment/gif/main.md) - End the capture of a gif, stored in `{Your Host OS desktop}/picotron_desktop`
