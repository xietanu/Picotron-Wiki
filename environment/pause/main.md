# Pause

## Overview

The pause menu is opened when you press `ENTER` or the `START`/`MENU` button (btn 6).
It contains the default options that Picotron provides alongside the cart-given options.

## Options

The default options consist of:

* Resume
* Favourite
* Reset Cart
* Exit

More options can be given through the use of [menuitem()](/picotron_api/functions/menuitem/main.md) by Picotron.

The default options are impossible to remove or change.
