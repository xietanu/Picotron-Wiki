# GFX Editor > Keybinds

## Overview

Information from here is sourced from the [Picotron Manual](https://www.lexaloffle.com/dl/docs/picotron_manual.html)

This goes over all the keybinds built into the GFX Editor for Picotron.

## Keybinds

### SPACE
Shortcut for the [Pan Tool](/environment/gfx_editor/tools/main.md)

### MOUSEWHEEL
Zoom in and out

### S
Shortcut for the [Select Tool](/environment/gfx_editor/tools/main.md) (hold down)

### CTRL-A
Select the entire canvas

### ENTER
Deselect everything

### ARROW KEYS
Move selection

### CTRL-C
Copy selection

### CTRL-V
Paste selection

### CTRL-B
Paste at 2x the size

### TAB
Toggle the RH pane

### -
Select previous sprite

### +
Select next sprite

### 1
Select previous color

### 2
Select next color

### RMB
Set the active color to the color of the hovered pixel

### F
Flip the selection horizontally

### V
Flip the selection vertically
