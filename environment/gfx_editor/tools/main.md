# GFX Editor > Tools

## Overview

Information from here is sourced from the [Picotron Manual](https://www.lexaloffle.com/dl/docs/picotron_manual.html)

This goes over all the tools built into the GFX Editor for Picotron, these tools are found below the palette and can be selected by clicking on them.

## Tools

### Pencil
Draw a single pixel

### Brush
Draw using a fill pattern and brush shape

### Line
Draw a line

Use `SHIFT` to snap closest axis, diagonal or 2:1 slope

### Rectangle
Draw a rectangle

Use `SHIFT` to snap to a square
Use `CTRL` to fill the rectangle

### Ellipse
Draw an ellipse

Use `SHIFT` to snap to a circle
Use `CTRL` to fill the ellipse

### Bucket
Fill an area with the active color

### Select
Select a rectangular region

### Pan
Change the camera position
