# load [options] path

## Overview

Loads `path` as a cartridge

Source: [source.lua](source.lua)

## Arguments

### [options]

#### `-b`

Open files from the workspace in the background, rather than stealing the focus of the current window

### path

The path, folder or file, to a cartridge

Append with `bbs://` protocol or `#` to load a cartridge from the bbs
