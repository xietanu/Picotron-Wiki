# export [outfile]

## Overview

Exports the loaded cartridge (`/ram/cart`)

Source: [source.lua](source.lua)

## Arguments

### outfile

File location to export

Supported filetypes to export to:

#### `.p64.png`

BBS Format for cartridges

#### `.html`

One-file web export

#### `.bin`

Binary exports for windows, linux and mac

## Future

Ability to export from any path, rather than `/ram/cart`
