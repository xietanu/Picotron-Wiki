# config [key] [value]

## Overview

`config` allows you to edit picotrons system settings (`/appdata/system/settings.pod`)

Source: [source.lua](source.lua)

## Arguments

### battery_saver `boolean`

Enable Picotron's battery saver

### blit_720p `boolean`

Use Picotron's 720p mode

### floppy_windows `nil`

Does absolutely nothing!

Left in by zep on accident?

No reference to floppy_windows in /system/

### fullscreen `boolean`

Fullscreen Picotron in the host OS

### mute_audio `boolean`

Mute the audio of Picotron

### network_access `boolean`

Allow Picotron to access your network

### pixel_perfect `boolean`

Whether Picotron should be scaled up to be pixel perfect

### rshift_magnify `boolean`

Whether pressing `RIGHT SHIFT` should magnify the area around your cursor

### snap_to-grid `boolean`

Files on the desktop snapping to a grid

### sparkles `boolean`

Sparkles follow the cursor

### stretch `boolean`

Stretch the Picotron screen to fill the entirity of your native screen

### swap_stereo `boolean`

Swap the sides of your stereo audio

### theme `string (filepath)`

Path to a .theme file

### wallpaper `string (filepath)`

Path to a picotron cartridge
