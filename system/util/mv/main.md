# mv [options] src dest

## Overview

Moves `src` to `dest`

Use `mv --help` for the built in instructions


Source: [source.lua](source.lua)

## Arguments

### [options]

Optional arguments for the command

### `-f`

overwrite `dest` instead of placing it inside of `dest`

### `-n`
no clobber, skips copying files if a file with the same name already exists in `dest`

### src

A folder or file to be moved

### dest

Location of the file to be moved to
